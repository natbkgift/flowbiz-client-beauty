# Pilot Feedback Interview - FlowBiz Beauty

Date: 2026-05-27
Scope: structured owner and staff feedback after friendly pilot

## Purpose

Use this interview at the end of a 14-30 day friendly pilot or during a conversion-readiness check. The goal is to capture workflow fit, trust, usability, value, risks, and willingness-to-pay signals.

Do not ask for unnecessary sensitive customer data during the interview.

## Owner Questions

- What problem did you hope FlowBiz would solve?
- Which workflow felt most valuable?
- Did you get better visibility into lead follow-up?
- Did the pilot reduce missed follow-up risk?
- Did no-show recovery become more systematic?
- Did repeat reminder opportunities become clearer?
- Did audit logs increase trust in staff/AI workflow?
- What still blocks paid onboarding?
- Which plan would fit if we continue?

## Staff Questions

- How often did you open the queue?
- Was approve/reject/modify clear?
- Were AI drafts useful?
- Which drafts did you edit most?
- Were any suggestions risky or off-brand?
- Did FlowBiz save time or add extra work?
- What screen or workflow confused you?
- What would make daily use easier?

## Willingness-To-Pay Indicators

Positive signals:

- Owner asks for next-month rollout.
- Staff used queue without prompting.
- Clinic requests workflow tuning.
- Clinic asks for user additions.
- Clinic asks for integration QA.
- Clinic compares pricing to recovered workflow value.

Weak signals:

- Usage happened only during check-ins.
- Owner cannot name a valuable workflow.
- Staff sees approval as extra work.
- Data handling cannot be agreed.
- Clinic only wants a custom CRM/EMR replacement.
