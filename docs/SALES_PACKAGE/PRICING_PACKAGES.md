# Pricing Packages - FlowBiz Beauty

Phase: 10 - Sales package

## Pricing Baseline

| Plan | Monthly Price | Target Clinic Type | Recommended Starting Point |
| --- | ---: | --- | --- |
| Starter | 9,900 THB/month | คลินิกเดี่ยวหรือทีมเล็กที่ต้องการลด lead หลุด | Friendly pilot หรือ clinic owner-led operations |
| Growth | 19,900 THB/month | คลินิกที่มีหลายช่องทาง lead และทีม sale/admin 2-5 คน | Most recommended MVP pilot plan |
| Pro | 39,000 THB/month | คลินิกที่มี volume สูง ต้องการ workflow, audit และ automation ที่จริงจัง | Multi-branch or high-volume clinic pilot |
| Enterprise | 80,000-200,000 THB/month | กลุ่มคลินิกหรือ chain ที่ต้องการ integration, governance และ support เพิ่ม | Custom pilot after technical discovery |

Prices are baseline packaging guidance. Final commercial terms should depend on pilot scope, integration requirements, support level, data migration, and compliance review.

## Feature Boundaries

### Starter

Best for:

- 1 clinic location
- Small staff team
- Basic lead follow-up workflow

Included in MVP positioning:

- Lead follow-up dashboard
- New Lead Welcome
- Uncontacted Lead Alert
- Basic HITL AI suggestion review
- Audit trail for key AI approval events
- Demo/simulated LINE and mock AI mode during early pilot setup

Not positioned as ready without setup:

- Real LINE send
- Real LLM generation
- Custom CRM integration
- Advanced BI

### Growth

Best for:

- Clinics with active Facebook/LINE lead flow
- Staff team that needs a daily operating queue
- Clinics that want repeat reminder workflow

Included in MVP positioning:

- All Starter items
- Lead Qualification Nurture
- No-Show Recovery
- Review Request
- Botox Cycle Reminder
- Filler Cycle Reminder
- Daily Marketing Reminder
- Pilot workflow tuning session

Integration status:

- Simulated by default for demo
- Real integration requires staging test plan, credentials managed outside repo, and no-real-send safety review

### Pro

Best for:

- Clinics with multiple staff roles
- Higher lead volume
- Audit and role control requirements

Included in MVP positioning:

- All Growth items
- Stronger RBAC/audit operating model
- More workflow tuning
- Pilot dashboard review
- Support for operational SOP design

Requires additional review:

- Branch-level reporting
- CRM/data import
- Production launch checklist
- Consent/PDPA process

### Enterprise

Best for:

- Clinic groups
- Multi-branch governance
- Custom integration and compliance needs

Included after discovery:

- Custom pilot plan
- Integration architecture review
- Governance/SOP workshop
- Staging-to-production readiness plan
- Dedicated support process

Not included by default:

- Full EMR replacement
- Full scheduling system
- Payment processing
- Inventory management

## Suggested Pilot Offer

Recommended pilot:

- 14-30 days
- 3-5 clinics total
- Growth plan as default pilot scope
- Pilot fee can be credited toward first paid month if conversion happens within the agreed window

Pilot deliverables:

- Demo tenant walkthrough
- Workflow setup for 3-5 core use cases
- Staff approval process
- Weekly metrics review
- Final pilot report with opportunity estimate

## Setup Fee Option

Suggested setup fee:

- Starter/Growth pilot: 15,000-30,000 THB
- Pro pilot: 40,000-80,000 THB
- Enterprise pilot: scoped after discovery

Setup fee may cover:

- Workflow mapping
- Demo/staging configuration
- Staff onboarding
- Template review
- Pilot reporting setup

## What Is Not Included

Unless explicitly scoped:

- Production deployment
- Real external send activation
- Real LLM provider activation
- CRM migration
- EMR integration
- Payment integration
- Inventory integration
- Full doctor scheduling
- Legal/PDPA consulting
- Paid ads spend management

## Discount Rule

Recommended discount guardrail:

- Discount only for limited pilot duration or annual prepay.
- Do not discount Enterprise before technical discovery.
- Do not trade away setup/support time without reducing pilot scope.
- Avoid promising business outcomes as a discount justification.

## Upgrade Triggers

Move from Starter to Growth when:

- Lead volume exceeds what one owner/operator can manage manually.
- Clinic wants repeat reminder and no-show workflows.
- More than one staff member needs approval queue access.

Move from Growth to Pro when:

- Multiple roles need stricter RBAC.
- Audit review becomes part of operations.
- Clinic needs more workflow tuning and reporting.
- Branch or manager review is required.

Move to Enterprise when:

- Multi-branch rollout is requested.
- Custom integration is required.
- Procurement/security review is required.
- Dedicated support or SLA is needed.
